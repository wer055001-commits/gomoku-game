# 更新歷程

## 2025-02-07

### 版本 1.2.0 - 增強最後落子標記
**更新內容：**
- ✨ 添加金色發光邊框，讓最後下的棋子更明顯
- 💫 添加持續脈衝閃爍動畫
- 🔴 增大中間標記點尺寸（從 8px 到 12px）
- ✨ 添加發光陰影效果

**改善原因：** 玩家反饋不容易看到對方剛下的棋子位置

---

### 版本 1.1.0 - 修復棋子閃爍問題
**更新內容：**
- 🔧 修復計時器更新時棋子閃爍的問題
- ⚡ 優化性能：計時器現在只更新時間顯示，不重新渲染整個頁面
- 🎮 改善遊戲體驗

**技術細節：**
- 新增 `updateTimerDisplay()` 函數，只更新 DOM 中的計時器元素
- 移除計時器回調中的 `render()` 調用
- 為計時器添加 `.black-timer` 和 `.white-timer` class

---

### 版本 1.0.0 - 初始版本
**功能特色：**
- 🎮 雙人對戰模式
- 🤖 AI 對戰模式
  - 簡單難度：適合新手
  - 中等難度：有一定挑戰
  - 困難難度：使用 Minimax 演算法，高手對決
- ↩️ 悔棋功能
- ⏱️ 計時功能（每位玩家 10 分鐘）
- 📜 歷史記錄
- 🔊 音效系統
  - 落子音效
  - 獲勝音效
- 🎨 精美的 UI 設計
  - 仿真木製棋盤
  - 漸變色棋子
  - 流暢動畫效果
- 📱 響應式設計，支援各種設備

**技術棧：**
- 純 HTML/CSS/JavaScript
- 無需依賴任何框架
- 單一檔案部署

## 部署資訊

**遊戲網址：** https://wer055001-commits.github.io/gomoku-game/

**GitHub 倉庫：** https://github.com/wer055001-commits/gomoku-game

**更新方式：**
修改代碼後，執行以下命令：
```bash
cd gomoku-game
git add .
git commit -m "更新說明"
git push
```
等待 1-2 分鐘後，更新會自動部署到網站上。
