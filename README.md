# 五子棋遊戲

完整功能的五子棋遊戲，包含 AI 對手、難度選擇、計時等功能。

## 功能特色

- ✨ 雙人對戰模式
- 🤖 AI 對手（簡單、中等、困難三種難度）
- ↩️ 悔棋功能
- 📜 歷史記錄
- ⏱️ 計時功能
- 🔊 落子音效
- 🎨 精美的 UI 設計
- 📱 響應式布局

## 安裝與執行

### 安裝依賴
```bash
npm install
```

### 啟動開發伺服器
```bash
npm start
```

遊戲將在 http://localhost:3000 開啟

### 建置生產版本
```bash
npm run build
```

## 遊戲規則

- 15x15 棋盤
- 黑子先行
- 先連成五子者獲勝
- 支援橫、豎、斜四個方向

## 技術棧

- React 18
- JavaScript ES6+
- CSS3
